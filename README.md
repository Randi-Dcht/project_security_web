# cybersecurity project

## Run project :

- Frontend :

> cd frontend
> 
> npm install
> 
> npm run dev

- Backend :

> cd backend/
> 
> symfony server:start

- for check security in backend:

> symfony check:security


## Information

- We will use `open ssl`
- client / server access (pc to connect)
- access on several machines
